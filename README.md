# Transaction Pattern Explorer

This project analyzes financial transaction data to identify spending patterns and unusual behavior.

## Features
- Data cleaning and preprocessing
- User-level spending metrics
- Visual analysis using seaborn and matplotlib
- Simple rule-based anomaly detection

## Tech Stack
- Python
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook
